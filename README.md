## Step 2

In your body, add a container and a child which will be your content

```
<body>
    <div class="container">
        <h1 class="content">
        Start of your sentence
            <ol>
                <li><span>Sentence 1</span></li>
                <li><span>Sentence 2</span></li>
                <li><span>Sentence 3</span></li>
                <li><span>Sentence 4</span></li>
            </ol>
        </h1>
    </div>
</body>
```
